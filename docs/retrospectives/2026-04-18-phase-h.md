## 격리 원칙 (v3.1 §3.2) 준수

- 외부 프로젝트마다 별도 Python subprocess (`asyncio.create_subprocess_exec`, shell=False).
- subprocess는 API 토큰 자체 보유 금지 — 메인이 `os.environ` 상속으로 주입, OS 종료 시 자동 정리.
- `max_concurrent=3` 세마포어 상한 — 초과 시 즉시 RuntimeError 반환.
- 요청 단위 spawn → MCP initialize → tools/call → 응답 반환 → 즉시 shutdown.
- subprocess 크래시는 `ToolResult(is_error=True)`로 격리 — 메인 loop 영향 없음.
